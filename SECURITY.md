# Security policy

## Reporting a vulnerability

Please do not disclose a suspected vulnerability in a public GitHub issue.

Send a concise report to `enukorahdevine@gmail.com` with:

- The affected URL or file
- Reproduction steps
- The expected and observed behavior
- Any relevant browser or device details

Do not include passwords, access tokens, private customer data, or other sensitive information.

## Scope

This repository hosts a static portfolio website. It does not provide authentication, payments, server-side forms, account storage, or a database.
